## 🎯 The Challenge

Help Phil optimize routes through **8 LA houses**, each with:
- ⏰ Time windows (9am-6pm slots)
- ⭐ Client preferences (1-10 scores)
- 🗺️ Travel distances

**Algorithm:** Dynamic Programming to maximize satisfaction!
